# Exemplos de SQL para Databricks

Coleção de consultas **genericas** (com dados fictícios).

## Como executar

- No Databricks, crie um notebook (SQL), copie e cole o conteudo de cada arquivo.
- Ajuste nomes de **schemas/locations** conforme seu ambiente.
- Todos os exemplos usam **dados fictícios**, sem qualquer referência a ambientes reais.
